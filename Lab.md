# Flipix Album - Lab Version Features

Based on a thorough exploration of the Flipix Album platform, here is a comprehensive list of all the features, tools, and sections available in the lab version:

## 1. Dashboard (Main Hub)
* **Album Analytics**: Displays the total number of albums created by the user.
* **Credit Status**: Shows the remaining album credits with a quick "Recharge" option for easy top-ups.
* **Remaining Days**: Displays the number of days left until the current subscription plan expires, including the exact expiry date.
* **Active Plan**: Shows current plan details (e.g., Free Credit) with an "Upgrade" button to seamlessly change plans.
* **Digital Album Preview**: An interactive 3D preview window showing a sample digital album (e.g., "Raja Weds Rani") with the following controls:
  * **3D Flip Effect**: Allows users to experience how the digital album looks and feels with page-turning animations.
  * **Background Music**: Play/Pause controls for the album's embedded music.
  * **Fullscreen Mode**: Expands the album preview to fill the entire screen for an immersive view.
* **Social Sharing Tools**: Quick-access floating buttons to share the album directly via WhatsApp, Instagram, Facebook, X (formerly Twitter), and LinkedIn.
* **Direct Support**: A floating "Chat with us" WhatsApp button available for real-time customer assistance.

## 2. Album Management
* **Create Album**: A detailed, step-by-step creation form featuring:
  * **Client Information**: Input fields for Client Name.
  * **Event Details**: Function Name (dropdown selection for predefined events like Wedding, Pre-Wedding, etc.) and a precise Date Picker.
  * **Music Selection**: Includes a searchable built-in library of songs, an "Upload Song" feature for custom tracks, and a volume control slider.
  * **Photographer Integration**: Allows selecting from a list of pre-saved photographers to auto-fill their contact details (Mobile, Address).
  * **Visual Content Upload**: Dedicated drag-and-drop sections for uploading the **Cover Front**, **Cover Back**, and multiple **Inner Sheets** of the digital album.
* **All Albums (List View)**: A centralized manager for all created albums with the following columns and tools:
  * **Album Details**: Serial Number, Client Name, Function Type, Date, and customized Song Name.
  * **Analytics**: Track total engagement/Views for each individual album.
  * **Tools**: 
    * **QR Code Generation**: Generate and download unique QR codes for each album.
    * **Link Copying**: One-click shareable URL copying.
    * **View Mode**: A dedicated "View" button to open and interact with the finalized digital album.
  * **Search & Filter**: A built-in search bar to quickly find specific albums by client name or event date.

## 3. Account & Partner Management
* **Your Photographers**: A mini-database to manage photography partners and vendors:
  * **Photographer List**: Displays Name, Mobile number, City, State, Address, and current Status in a tabular format.
  * **Management Tools**: Functionality to Add New Photographers, Edit existing partner details, or Delete outdated profiles.
  * **Search functionality**: Quickly filter and find photographers by name or city.
* **Recharge History**: A detailed billing, transaction, and credit tracking section:
  * **Plan Summary**: Shows the active plan name, start and end dates, and overall payment status.
  * **Credit Monitoring**: A visual progress bar depicting used versus total available credits.
  * **Transaction History**: A searchable and sortable table documenting all past transactions (Transaction ID, Amount Paid, Credits added, Date, etc.).

## 4. Support & Resources
* **FAQ (Frequently Asked Questions)**: A categorized knowledge base answering common questions about Flipix Album, platform specific operations, and subscription/billing plans.
* **Contact Us**: A direct communication page for reaching out to the support and technical teams via forms or provided contact details.

## 5. Global Interface & Usability Features
* **Header Bar**: Features a quick "Create Album" primary action button, an "Album Left" live credit indicator, a Notifications bell, and customizable Profile settings.
* **Collapsible Sidebar**: A clean, modern navigation menu that can be toggled to maximize workspace, allowing easy movement between Main dashboards, Account settings, and Support sections.
* **Responsive Design**: The interface adapts to provide a seamless management experience across various screen sizes.
